## オブジェクト

オブジェクトですよ奥さん。

オブジェクトというのは配列みたいだけど、自分の好きな名前で管理できるものです


例
```
$player->sendMessage("Hello");
```

というもんです。

この中にPlayer名、Opか否か、表示名、ゲームモードなどが入っています。

この場合は$playerはただの変数ではありません。

なぜオブジェクトが必要かというと、これはオブジェクト指向のお話になります。興味があるなら調べてみては？

簡単に言うと、製作者が作りやすいからです。(超アバウト)

本当は説明がいっぱいあるけれど(カプセル化とか継承とか...)、plugin開発にはこれだけあれば十分かな。

とりあえずものは知っておいてほしいのでやりました。

### コメントについて

忘れてました。(;´∀｀)テヘッ

プログラムの中は""の中以外は半角英数字しか使えないという決まりです。

しかし、プログラムをわかりやすくするために説明などが書きたい場合があります。

その時に使うのがコメントです。


```
//←１行の場合はこのように書いてそのあとがコメントで、

/*←このように書くと

複数行でも大丈夫

間がコメント。→*/

注意：/**/で１セットです/**/の間がコメント。
```

オブジェクトは本気でプラグイン作るなら必要ですが、ぬるーくやるんだったら作らなくてもいいです。(使わないとは言っていない)

逆に本気で作りたいなら必要です。
